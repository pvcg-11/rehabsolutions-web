Un sistema mecatrónico basado en impresión 3D, servomotores, sensores y control inteligente mediante ESP32

ESP32
Servomotor MG996R

Proporciona movimientos asistidos precisos y controlados para la rehabilitación

Sensores avanzados

Sensores de flexión y presión para medir esfuerzo y progreso en tiempo real

Control ESP32 y PWM

Microcontrolador potente con modulación PWM para control preciso del motor

Impresión 3D
